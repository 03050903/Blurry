Blurry is an easy blur library for Android.

Screenshot

How do I use it?

Setup

Gradle
dependencies {
    compile 'jp.wasabeef:blurry:x.x.x'
}

Functions

Overlay

Parent must be ViewGroup

Blurry.with(context).radius(25).sampling(2).onto(rootView);

Into

Blurry.with(context).capture(view).into(imageView);

Blur Options

  • Radius
  • Down Sampling
  • Color Filter
  • Asynchronous Support
Blurry.with(context)
  .radius(10)
  .sampling(8)
  .color(Color.argb(66, 255, 255, 0))
  .async()
  .onto(rootView);

Requirements

Android 2.3+
